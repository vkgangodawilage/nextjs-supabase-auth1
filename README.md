## Getting Started

```bash
git clone https://github.com/vkgangodawilage/nextjs-supabase-auth1.git
cd nextjs-supabase-auth1
npm install
```

## Environment Variables

Create `.env.local`:

```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```
